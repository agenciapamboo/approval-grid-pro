# Deploy Manual via Lovable Dashboard

## Passo 1: Acessar Lovable
1. Acesse: https://lovable.dev
2. Faça login
3. Abra o projeto "approval-grid-pro"

## Passo 2: Forçar Novo Deploy
1. No dashboard do projeto, procure por:
   - **"Deploy"** ou **"Deployments"** no menu
   - Botão **"Redeploy"** ou **"Deploy Now"**
   
2. Clique em **"Deploy from main"** ou **"Trigger Deploy"**

## Passo 3: Aguardar Build
- O Lovable vai fazer build automaticamente
- Aguarde 2-5 minutos
- Você verá um log de progresso

## Passo 4: Verificar Deploy
- Quando completar, aparecerá um link de preview
- Clique no link ou acesse sua URL de produção
- Faça **Hard Reload** (Cmd+Shift+R)

## Passo 5: Testar
1. Abra o console (F12)
2. Teste o assistente de IA
3. Verifique se o dialog bonito aparece com 3 sugestões

---

Se não encontrar a opção de deploy manual no Lovable, vá para OPÇÃO 2 (Build Local).
